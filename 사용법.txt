═══════════════════════════════════════════════════════
  RFP 제안서 작성기 사용법
═══════════════════════════════════════════════════════

[ 첫 실행 ]
  1. "제안서작성기실행.bat" 더블클릭
  2. 처음 한 번만 Python 및 필요 패키지가 자동 설치됨 (5~10분)
     - 인터넷 연결 필요
     - 약 500MB 디스크 공간 사용
  3. 설치 후 자동으로 브라우저(http://localhost:8501)가 열림

[ 두 번째부터 ]
  - "제안서작성기실행.bat" 더블클릭 → 바로 실행

[ 종료 ]
  - 검은 콘솔창 닫기 (Ctrl+C 또는 X 버튼)

═══════════════════════════════════════════════════════
  주요 기능
═══════════════════════════════════════════════════════

1. RFP PDF 업로드
   - 요구사항 자동 추출
   - 목차(장표) 자동 구성

2. 전략 수립 (장표 작성 탭)
   - 챕터 선택 후 "전략 수립 시작"
   - 사전 의견 입력 → 전략가 vs 비평가 토론 (N라운드)
   - 추가 의견/참고 문서 반영 → 최종 전략 정리
   - 필요 참고문서 자동 식별 → 업로드 (페이지 지정 시 VLLM 파싱)

3. 장표별 초안 작성
   - 목차(소제목 + scope) 자동 생성
   - 소제목별 개별/일괄(병렬) 본문 생성
   - 챕터 전체 마크다운 다운로드 (Claude/GPT에 PPT 변환 요청 가능)

═══════════════════════════════════════════════════════
  API 키 변경
═══════════════════════════════════════════════════════

  - 폴더 안 ".env" 파일을 메모장으로 열어 OPENAI_API_KEY=... 부분 교체
  - 저장 후 재실행

═══════════════════════════════════════════════════════
  문제 해결
═══════════════════════════════════════════════════════

  - 브라우저가 안 열림: 콘솔창에 표시된 주소를 수동으로 복사해서 접속
  - "Already configured" 후 작동 이상: python 폴더 삭제 후 재실행
  - PDF 파싱 오류: PDF가 스캔본(이미지)일 수 있음 → 텍스트 기반 PDF 필요
